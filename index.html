How I Built This Website on My Phone

I built this website completely from my mobile phone using simple tools available for Android/iOS. Below is a detailed guide on how I did it, so you can do it too:

  1. 📲 I installed the GitHub app and created an account.
  2. 📝 I opened my browser and went to github.com to create a new repository.
  3. 💻 I wrote the code in GitHub's code editor (or you can use VS Code Web).
  4. 🚀 I saved the file as index.html and committed the changes.
  5. 🌐 I went to Settings → Pages → Enabled GitHub Pages using the main branch and root folder.
  6. ✅ My site was live at: https://kuuriya.github.io

If you want to learn more or copy this code, visit my GitHub repository.
